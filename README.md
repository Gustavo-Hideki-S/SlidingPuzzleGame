# 🧩 Sliding Puzzle Game

A simple sliding puzzle game developed in **Java** using the JSGE engine.

---

## 📖 About

This project is a simple implementation of a sliding puzzle game built with Java.  
It demonstrates core concepts such as:

- Game loop handling
- Rendering
- User input
- Puzzle logic

The game was developed using **JSGE**, created by  
[David Buzatto](https://github.com/davidbuzatto).

---

## ⚙️ Technologies

- ☕ Java  
- 🎮 JSGE
  
---

## ▶️ How to Run

> [!IMPORTANT]
> This project was developed using NetBeans.  
> The JSGE engine is configured to run within NetBeans, so if you want to run it in another IDE, make sure to copy only the source code or properly configure the environment.
